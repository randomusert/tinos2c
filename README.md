# Tinos2 ce
Tinos2 ce is a build of tinos2 writen in C as in ce standing for C edition, not community edition.

Unlike it's precessor tinos2 ce is built from already existing code writen by me.

So come take a look at the os.
## building
to build:

``` bash
# requires clang, nasm and ld
make

```
## running
to run, you have 2 options:

1. ``` bash
   make run
   ```
2. ``` bash
   qemu-system-i386 -cdrom tinos-c.iso # or qemu-system-x86_64 -cdrom tinos-c.iso
   ```
